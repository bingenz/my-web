Cau hoi thuong gap

Phan nay giup giam so lan khach hoi lap lai va giu nguoi xem o lai trang lau hon truoc khi bam lien he.

Toi co the doi toan bo noi dung sau nay khong?

Co. File duoc viet theo huong de sua text truc tiep. Ban co the thay tieu de, gia, mo ta, nut bam va cac section bang cach sua HTML thong thuong.

Trang nay co phu hop de encode sang file xuat ban khong?

Co. Cau truc van la mot file HTML day du, tuong thich voi luong `encode-html.js` dang co trong project.

Neu toi muon chen link Zalo, Telegram hay form dat hang?

Rat de. Chi can thay `href` cua cac nut CTA hoac them mot khoi form o cuoi trang la duoc.

Co comment an hay ghi chu nao con sot lai khong?

Khong. Ban rewrite nay duoc viet lai sach, khong chen note comments kieu `*`, `#` hay comment HTML/CSS/JS.
